Place the official Kick badge images in this folder. Use lowercase filenames that match the badge
type, for example:

- verified.svg
- moderator.svg
- vip.svg
- staff.svg
- bot.svg
- founder.svg
- partner.svg
- broadcaster.svg

Any of the listed files can use .svg, .png, .webp, .gif, .jpg, or .jpeg extensions. The application
will automatically pick the first matching file it finds for each badge.
